git init: khởi tạo local repository
git add <tên file>: đẩy file demo.txt vào stagging area
git add . : đẩy tất cả file vào stagging area
git commit -m <message "hello"> : đưa thay đổi vào local repository, tạo 1 snap shot lưu trữ online lịch sử thay đổi
git remote: kết nối local repo với remote repo


Quy trình: file -> stagging area (lưu tạm) -> local repository (kho offlinegit comm) -> remote repository (kho online)
